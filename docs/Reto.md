# Reto

## Consideraciones

Para esta prueba he documentado la **Configuración de un segundo factor de autenticación (2FA) con una app de autenticación (TOTP)**. Es una tarea concreta, con principio y fin claros, y de dificultad media-alta, lo que permite mostrar distintas habilidades de redacción técnica.

He incluido una sección de problemas frecuentes para responder a dudas habituales durante el uso. Está pensada para demostrar capacidad para anticipar incidencias y orientar al usuario.

## Herramientas y referencias

!!! info "Visualización óptima"
    Para ver el tutorial con todas sus funcionalidades y su diseño completo, puede consultarse [la versión publicada en GitHub Pages](https://miguelolmedomorell.github.io/guardedbox-test/).

La página está creada en un entorno **MkDocs** (Markdown + flujo Git) y publicada en **GitHub Pages**. He elegido este enfoque porque refleja varias habilidades relevantes para el puesto:

- Dominio de Markdown, incluidas funciones avanzadas específicas de algunas plataformas.
- Familiaridad con flujos de trabajo basados en Git.
- Capacidad para crear una plataforma de documentación desde cero.

Para ello, he usado:

- mkdocs versión 1.6.1
- Tema mkdocs-material versión 9.7.5
- Python versión 3.14.3

Como base lingüística he seguido la [Guía de estilo de escritura de Microsoft](https://learn.microsoft.com/es-es/contribute/content/style-quick-start). También he tomado como referencia ejemplos reales de documentación de:

- [Adobe](https://helpx.adobe.com/la/manage-account/using/secure-your-adobe-account.html)
- [Apple](https://support.apple.com/es-es/102660)
- [GitHub](https://docs.github.com/es/authentication/securing-your-account-with-two-factor-authentication-2fa/configuring-two-factor-authentication)
- [Bitwarden](https://bitwarden.com/es-la/help/setup-two-step-login-authenticator/)

Para las capturas de pantalla utilicé la **Herramienta Recortes** de Windows, por su sencillez y accesibilidad.

He utilizado el color naranja para resaltar elementos clave en las imágenes, al resultar menos agresivo que el rojo según las recomendaciones de [Fluent 2 de Microsoft Design](https://fluent2.microsoft.design/color).

Por último, he intentado ajustar la paleta de colores al estilo de GuardedBox dentro de las posibilidades del entorno y he añadido el logotipo corporativo como favicon para reforzar el acabado profesional.
